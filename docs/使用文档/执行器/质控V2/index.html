质控引擎

简述​

  • 质控使用 source -〉measure -〉 sample -〉 sink 的流程。
  • source支持：mysql，hive
  • measures支持：记录数、空值校验、范围校验、值域校验、格式校验、重复校验、最大值、最小值
  • sink支持：api
  • sample支持：MinIO

使用​

export SPARK_MAJOR_VERSION=2;
spark-submit --class com.xx.spark.qc.Application 
--master yarn 
--deploy-mode client  
spark-quality-control-executor.jar <taskID> <taskInfo>

Note: taskInfo 为任务json的base64

编译​

cd spark-quality-V2-executor
mvn clean package -DskipTests assembly:assembly

基本参数​

参数是否必须类型说明
id是String任务ID
name是String任务名称
engine是String引擎类型，暂支持离线。默认需为batch
source是Object配置，见下
measures是Array配置，见下
sink是Object配置，见下

source​

支持：mysql， hive

mysql​

参数是否必须类型说明
sourceType是String默认mysql
url是Stringjdbc连接url，如jdbc:mysql://xx:3306/test
user是Stringmysql用户
passwd是String密码
table是String表名
partition否String分区，如 dt=202201
group否String分组
condition否String条件，以sql的where条件为准。如： code is not null

hive​

参数是否必须类型说明
sourceType是String默认hive
databases是String数据库
table是String表名
partition否String分区，如 dt=202201
group否String分组
condition否String条件，以sql的where条件为准。如： code is not null

measures​

支持类型如下：

类型级别code说明
记录数表级别records
空值校验字段级别empty
范围校验字段级别numRange
值域校验字段级别dictRange
格式校验字段级别regular
重复校验字段级别duplicate
最大值字段级别max
最小值字段级别min

记录数 - records​

{
  "id": "50",
  "measuresType": "records",
  "level": "table",
  "options": {
  },
  "threshold": {
    "range": [20,30]
  }
}
参数是否必须类型说明
id是String任务ID
measuresType是Stringrecords
level是Stringtable
options是Object默认为空
threshold是Object合格阈值
threshold.range是Array[Int]范围，  Array[0]为最小值，Array[0]为最大值
